## Sponsor Technologies (relevant to this hackathon)

### 0G — Decentralized AI Infrastructure

- **Funding:** $300M+
- **Core:** Decentralized storage + compute layer for AI agents
- **ERC-7857 (iNFT standard):** Extends ERC-721 with encrypted metadata for tokenizing AI agents
  - Agents become tradeable, ownable assets on-chain
  - Owners receive earnings and gains generated by their agent
  - Secondary market royalties for agent creators
  - [Docs](https://docs.0g.ai/developer-hub/building-on-0g/inft/erc7857)
- **Web 4.0 marketplace:** Launched with AIverse (March 2026) — AI agents can own, trade, and evolve on-chain
- **OpenClaw:** Open agent framework referenced by 0G Track A (forks/extensions welcome)

### Gensyn — Decentralized Compute + AXL

- **AXL (Agent eXchange Layer):** P2P communication primitive
  - No central server — uses Yggdrasil (decentralized IPv6 overlay)
  - End-to-end encrypted, application-agnostic
  - Runs as a local node; agents talk to it over localhost
  - Use cases: distributed inference, collective training, multi-agent swarms, autonomous system comms
  - [Docs](https://docs.gensyn.network)
  - [Intro post](https://blog.gensyn.ai/introducing-axl/)
- **Mainnet:** Live as of April 22, 2026
- **Delphi:** First native app — permissionless market where AI systems generate and verify outcomes
- Grant pipeline: all hackathon winners fast-tracked into Gensyn Foundation grants

### ENS — Agent Identity Layer

- **ENSIP-25:** Standardized way to verify an AI agent's ENS name against an on-chain registry (ERC-8004)
  - Defines a specific ENS text record format linking registry entry to name
  - Deterministic verification without new contracts or resolver upgrades
- **ENSIP-26:** ENS Native AI Identity proposal
- **Key use cases for agents:**
  - Human-readable identity across chains (Ethereum, Base, Optimism, etc.)
  - Payment signaling: publish how an agent expects to be paid (x402 + ENS)
  - Reputation layer: positive/negative reputation consistent across app contexts
  - Access gating, metadata storage, agent discoverability
- **Why it matters:** No credible way to create and verify agent identity currently — ENS fills this gap

### KeeperHub — Execution Layer

- Onchain execution infrastructure for autonomous agents
- Bridges payment rails and agent frameworks
- Feedback-friendly: $500 bounty just for actionable bug/DX reports

## Broader Ecosystem

|----------|---------|

### Payment Standards

| Standard | Purpose |
|----------|---------|
| x402 | Machine payments — agents pay for API calls, data, compute without accounts |

x402 was developed by Coinbase and launched alongside Agentic Wallets (February 2026). Enables account-free commerce with no subscriptions or minimum fees.
