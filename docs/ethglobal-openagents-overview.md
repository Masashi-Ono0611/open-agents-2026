## What to Build

The hackathon targets projects at the intersection of:

- **AI agents** — autonomous, goal-driven agents; multi-agent swarms
- **Onchain infrastructure** — execution layers, agent frameworks, tooling primitives
- **DeFi** — agents that swap and settle value onchain
- **Identity** — agent identity, discoverability, and coordination via naming systems

Core theme: AI agents that operate in decentralized environments with real onchain execution — not demos, but working systems.

## Judging Process

- **Round 1 (async):** All submitted projects are reviewed asynchronously
- **Round 2 (live):** Top ~20% of projects advance to live judging
- Both rounds use the same judging criteria
- Judges emphasize **real utility** over novelty

---

## Submission Requirements

All projects must include:

1. Functional demo (live or recorded, 2–4 minutes)
2. Public GitHub repository with README
3. Team contact information
4. Clear documentation of approach and integration depth
5. Select up to **3 Partner Prize** tracks to apply for
